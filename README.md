# Вожаҷӯи Шоҳнома 🏰

Ин бозии интернетии "Вожаҷӯи Шоҳнома" мебошад, ки бо мақсади омӯзиш ва эҳёи калимаҳои қадимаи забони порсӣ аз "Шоҳнома"-и Абулқосим Фирдавсӣ сохта шудааст. 

## Технологияҳои истифодашуда:
- **Фронтенд:** Vue.js 3, Tailwind CSS, Pinia, Vue Router
- **Бэкенд:** Node.js, Express.js
- **Махзани маълумот:** MongoDB

## Тарзи ба кор андохтани лоиҳа:

Барои ба кор андохтани лоиҳа дар компютери худ, қадамҳои зеринро иҷро кунед:

### 1. Талаботи пешакӣ (Prerequisites)
- [Node.js](https://nodejs.org/) бояд дар компютери шумо насб шуда бошад.
- [MongoDB](https://www.mongodb.com/try/download/community) бояд насб ва фаъол бошад (Дар порти `27017`).

### 2. Ба кор андохтани Бэкенд (Сервер)
1. Ба папкаи `backend` дароед:
   ```bash
   cd backend
   ```
2. Модулҳои лозимаро насб кунед:
   ```bash
   npm install
   ```
3. Серверро ба кор андозед:
   ```bash
   npm start
   ```
   *(Сервер дар порти `5001` ба кор медарояд)*

### 3. Ба кор андохтани Фронтенд (Бозӣ)
Дар як терминали нав (алоҳида) қадамҳои зеринро иҷро кунед:
1. Ба папкаи `vozhaju-vue` дароед:
   ```bash
   cd vozhaju-vue
   ```
2. Модулҳои лозимаро насб кунед:
   ```bash
   npm install
   ```
3. Бозиро ба кор андозед:
   ```bash
   npm run dev
   ```
   *(Бозӣ дар суроғаи `http://localhost:5173` кушода мешавад)*
